JS中class的使用

一，概念

  1. 类是用于创建对象的模板。JavaScript 的原型可以解释为对一种面向对象的概念，类似于一种非正式的类（class）。
  2. class 表示法提供了一个显式方法，来定义一个构造函数及其原型。
  3. 类可以指明一个object的属性和方法，这个对象就称为类的实例。（给定一个对象和一个构造函数，instanceof 运算符可以告诉你该对象是否是该构造函数的一个实例）

二，定义class

类是特殊的“函数”，和function类似，class也有两种形式：类声明（statement）和类表达式（expression）。

1.类声明

  • class 关键字是类声明的开始，它允许我们在一个地方定义一个构造器和一组方法。

  • 任何实例化的对象都可以调用。例如，这里任何new Rabbit都可以调用 killerRabbit.speak(“123”)。

  • 可以在类声明的大括号内写入任意数量的方法（method），而constructor（构造函数）是一个特殊的方法，这种方法用于创建和初始化一个由class创建的对象。一个类只能拥有一个名为 “constructor”的特殊方法。

  • 类声明目前只允许method函数添加到原型中。 在那里保存一个非函数值可能会有点不方便。目前，你可以在定义该类后直接操作原型来创建这些属性。例如，abc=”123”不行，操作原型：Rabbit.prototype.abc=”123”

  • 类声明和函数声明之间的一个重要区别在于, 函数声明会提升，类声明不会提升。所以首先需要声明类，然后再访问它。

2.类表达式

  • 类表达式可以命名或不命名。命名类表达式的名称是该类体的局部名称。

三，Getters, setters, and statics

  1. 实例的属性必须定义在类的方法里。
  2. 静态的或原型的数据属性必须定义在类定义的外面。
  3. static 关键字用来定义一个类的一个静态方法。调用静态方法不需要实例化该类。
    Temperature class allows you to write Temperature.fromFahrenheit(100)

四，extends扩展子类，super调用超类

  • extends这个词用于表示这个类不直接基于默认的Object原型，而是基于其他类。这被称为超类（superclass）。派生类是子类（subclass）。
  • 构造函数使用super 关键字调用其父类的构造函数。（set方法里用到了super.set,也是获取父类的set方法）
  • 如果子类中定义了构造函数，那么它必须先调用 super() 才能使用 this。
