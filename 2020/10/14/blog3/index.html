《关于昨天代码上传失败的原因》

错误1：没把代码所在的文件复制到git clone生成的lishuminer.github.io文件里
错误2：没有在开始的git bash输命令打开生成的文件夹，而是在生成文件夹处自行打开git bash

上传代码的步骤：

  • 在要上传的文件夹处git bash,输入命令:git clone https://github.com/lishuminer/lishuminer.github.io.git
  • 打开此文件夹，将其中的代码文件复制到生成的lishuminer.github.io文件夹中
  • 继续在git bash中用命令行cd lishuminer.github.io (打开这个文件夹）
  • git add . 不要忘记“.”前面的空格！
  • git commit -m “项目说明” 随便填一个说明
  • git push -u origin master
