《学而和ins页面总结》

一 整体布局:

  1. 大盒子:header,main,footer
  2. 大盒子里有版心盒子,确保内容居中
    `
    ...

.header {
height: 57px;
}
.center {
width: 900px;
height: 57px;

margin: 0 auto;
}
3. 设置消除浏览器默认间隙

  • {
    margin: 0;
    padding: 0;
    }
    `

二 css局部细节

  • 用 position:fixed 固定不滚动的部分

  • 消除搜索框和搜索按钮间隙用float
    .search { float: left; width: 200px; height: 25px; outline: none; } .btn { float: left; width: 75px; height: 25px; }

  • 横向滚动用overflow
    .gun { height: 125px; justify-content: space-between; white-space: nowrap; //不换行 overflow-y:hidden; overflow-x:scroll; //横向滚动 }

  • 尽量不使用绝对定位(脱离文档流)

  • 关于hover

  1. 学而的首页,li,课程动态,兔子部分:
    • 字变红且出现上边框:
      鼠标没有移上去时:设置一个透明上边框border-top: 4px solid transparent(防止hover时字被上边框挤下去);
      鼠标hover时:
      .l:hover { border-top: rgb(244, 67, 54) solid 4px; color: rgb(244, 67, 54); }
    • li第二行设置了灰色,会覆盖hover时的红色 解决办法:
      `
      .line li:hover {
        color: rgb(244, 67, 54);
      }
      .line li:hover > span {
        color:  rgb(244, 67, 54);
      }
      `
    • 背景颜色变深:
      .book {
      background-color:rgba(247, 123, 114); //浅红
      }
      .book:hover{
      background-color:rgba(244, 67, 54); //深红
      }
    • 鼠标移到.book时兔子出现:
      .redtutu {
      position: absolute; //相对于.book绝对定位
      width: 80px;
      margin-left: -97px;
      margin-top: -48px;
      visibility: hidden;
      }
      .book:hover > .redtutu { //book和兔子用子代选择器>
      visibility: visible;
      }
  2. ins的gmmtv部分:
    鼠标没有移上去时.tv隐藏;鼠标移动到.h范围时.tv的属性变为显现(这里.h和.tv为兄弟关系,用
    +来连接)

.tv { visibility: hidden; } .h:hover + .tv { visibility: visible; }
